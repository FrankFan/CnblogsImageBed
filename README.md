CnblogsImageBed
===============

CnblogsImageBed——A place to store images.

## 功能
* <del>登录博客园</del>
* <del>上传图片</del>
* <del>获得图片外链地址</del>
* 记住登陆账号7天、15天等
* Drag&Drop,拖拽式上传
* 后续继续优化
